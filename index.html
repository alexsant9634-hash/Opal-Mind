Funcionalidades

Tudo que você precisa para pensar melhor

Canvas infinito

Espaço ilimitado para suas ideias crescerem. Zoom, pan e navegação fluida em qualquer dispositivo.

Edição intuitiva

Duplo clique para editar, Tab para criar filhos, Enter para irmãos. Sem curva de aprendizado.

Salvo na nuvem

Todos os seus mapas salvos automaticamente. Acesse de qualquer dispositivo, a qualquer hora.

Modelos prontos

SWOT, Brainstorm, Plano de Projeto, Revisão Semanal e mais. Comece em segundos.

Exportar e compartilhar

Exporte como PNG ou JSON. Apresente seu mapa em modo tela cheia para qualquer audiência.

Layouts automáticos

Organize em árvore horizontal, vertical ou radial com um clique. Seu mapa sempre organizado.

Como funciona

Do pensamento ao mapa em minutos

01

Crie sua conta

Cadastro rápido com e-mail e senha. Sem cartão de crédito, sem complicação.

02

Novo mapa

Crie do zero ou escolha um modelo pronto. O nó central já está esperando por você.

03

Expanda suas ideias

Use Tab e Enter para criar ramos. Arraste, reorganize, adicione notas, tags e links.

04

Salvo automaticamente

Cada alteração salva na nuvem em tempo real. Volte quando quiser, tudo estará lá.

Configurar Supabase
Para habilitar autenticação e salvar seus mapas, conecte ao Supabase.
create table public.mind_maps ( id uuid default gen_random_uuid() primary key, user_id uuid references auth.users(id) on delete cascade not null, title text not null default 'Mapa sem título', data jsonb not null default '{}', created_at timestamptz default now(), updated_at timestamptz default now() ); alter table public.mind_maps enable row level security; create policy "crud_own" on public.mind_maps for all using (auth.uid() = user_id) with check (auth.uid() = user_id);
Usar sem conta (modo offline)
